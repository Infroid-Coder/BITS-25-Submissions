ConsolePastel

ConsolePastel is a developer utility that allows styled and colored console outputs using ANSI escape codes. It provides easytouse functions like format, log, and f for applying styles and colors to specific parts of text or entire strings. Designed for both Node.js and web environments, it simplifies debugging and enhances the readability of terminal output for developers.
